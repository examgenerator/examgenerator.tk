¡Bienvenido a Exam Generator!

Exam Generator es una aplicación web simple para preparar exámenes muy rapídamente.

Nuestro concepto:

Preparar exámenes puede tomar mucho tiempo. Cuándo quiere evitar la trampa, puede preparar dos o tres versiónes. ¿Pero puede ud. hacer versión para cada estudiante rapídamente? ¡Sí, con Exam Generator!

Algunas cosas que ud. puede hacer con Exam Generator:

  • Preparar exámenes del vocabulario
  • Elegir una pregunta aleatoria
